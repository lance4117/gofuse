# Go Starter Kit

一个轻量、模块化、可扩展的 Go 项目启动框架，内置常用业务组件的封装，包括 Gin、Redis、数据库、配置管理、日志系统等，适合用于快速启动中小型后端项目。

> 🛠️ 本项目旨在锻炼开发者的工程实践能力，构建一套贴近真实业务的项目脚手架。

---

## ✨ 功能特性

- 🔧 Gin HTTP 框架封装
- 🧱 Redis 客户端初始化与封装
- 💾 MySQL 数据库连接管理
- ⚙️ 多环境配置加载（基于 viper）
- 📜 模块化目录结构
- 🧩 Gin 中间件支持（日志、错误恢复等）
- 📦 工具函数库拓展支持
- ✅ 单元测试/集成测试结构预留
